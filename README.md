# Introduction to HTML and CSS. A Demo Project

## Overview

This project is a basic demonstration of common HTML elements styled lightly with CSS. It shows how different parts of a webpage work, using only HTML and CSS, without any JavaScript.

The goal is to help beginners understand the structure of a webpage and see how user-interface elements look and behave.

## User Guide

- Open index.html in any browser or follow this link to a deployed app.
- Interact with the elements.
- Explore how each behaves.
